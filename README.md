# d365-dualwrite-toolkit

Open-source D365 Dual-Write entity checker, sync planner & go-live toolkit.

## What this does
- Validate Dual-Write entity mappings and configuration
- Generate sync plans and pre-flight checklists
- Provide guidance for go-live and troubleshooting

## Features
- Entity and field validation
- Sync dependency planner
- Exportable reports and checklists

## Installation
Clone the repository and follow tool-specific docs in /docs (if present):

git clone https://github.com/gaurangjani/d365-dualwrite-toolkit.git

## Usage
See the documentation and examples in the repository. Typical workflow:
1. Run the entity checker to find mismatches
2. Generate a sync plan
3. Review the go-live checklist and export reports
